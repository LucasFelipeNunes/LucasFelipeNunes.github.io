Habilidades

Linguagens de Marcação e Estilo

As linguagens de marcação e de estilo (ou de folha de estilo) são aquelas que regram a apresentação visual e auditiva de documentos estruturados e de seus textos.

Atualmente, eu trabalho com as linguagens de marcação HTML5, LaTeX e Markdown, e com a linguagem de estilo CSS3.

Linguagens de Programação

As linguagens de programação são aquelas que criam uma estrutura sintática e semântica para os programadores enviarem instruções para o computador.

Atualmente, eu tenho conhecimentos base nas linguagens de programação Assembly, Código G (G-code), R, C, C#, C++, Java e JavaScript.

Ambientes de Desenvolvimento

Os ambientes de desenvolvimento são espaços de trabalho que permitem aos desenvolvedores criarem ou fazerem alterações no código de um programa sem afetar sua versão ativa.

Atualmente, eu trabalho com os softwares Dev-C++ (para as linguagens C e C++), Geany (para a linguagem Java), Visual Studio (para C#) e Visual Studio Code (para outras linguagens), além das IDEs de R e Arduino.
